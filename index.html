Моё резюме   

Привет! Меня зовут Наталья. Я начинающий Frontend-разработчик.

Сильное желание стать программистом возникло у меня 2 года назад. Профессия программиста привлекла меня тем, что можно и нужно всё время учиться и развиваться. Активным же изучением web-разработки я занялась в октябре 2015. Прошла очные курсы от Центра IT-образования по специализации WEB Developer.

На курсах были получены знания:

  • HTML и CSS
  • Vagrant
  • JavaScript
  • Nginx
  • Bootstrap
  • SQL (SQLyog)
  • Git
  • PHP

Недавно я прошла стажировку в eLama. Три месяца с 17 апреля по 17 июля мы командой стажеров разрабатывали проект с нуля: от подготовки MVP, проектирования архитектуры и выбора технологий до самой реализации. К концу стажировки, за две недели разрабатывали индивидуальные проекты (Save Money/продемонстрирован ниже).

На стажировке мною использовались:

  • Bootstrap
  • JavaScript
  • Docker
  • Ajax
  • Git
  • jQuery
  • Handlebars.js
  • npm
  • Twig
